Project Neon

Description

Project Neon is a 2.5D platformer that focuses on movement, responsiveness and level design. The aim is to complete the levels as quickly as possible. There are various techniques you can use to beat your own record time.

Meine Aufgaben

  • Gameplay Programming
  • Implementing basic Movement
  • Leveldesign
  • Implementing the Timer

Technologies
